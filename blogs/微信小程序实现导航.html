    微信小程序实现导航

    xin 4/11/2022 微信小程序

    这种页面的切换效果是通过左右滑动进行切换的，在内容区域默认显示的音乐推荐的页面，用鼠标向左滑一下，播放器页面就会显示出来，再往左滑就会把播放列表显示出来，像这种效果类似于网页中的焦点图的效果，在tab区域有3个按钮0、1、2，如果当前位于info.wxml页面，0这个按钮就是点亮的样式，也可以通过点0、1、2这几个按钮来切换到对应的标签页。 swiper组件是小程序里面的滑块视图容器组件，这个组件经常用于实现轮播图，在这里不仅要做轮播图，还要实现标签页切换效果

    swiper 组件常用属性
indicator-dots	        Boolean	是否显示面板指示点，默认为false
indicator-color	        Color	指示点颜色，默认为rgba(0,0,0,.3)
indicator-active-color	Color	当前选中的指示点颜色，默认为#000000
autoplay	            Boolean	是否自动切换，默认为false
current	                Number	当前所在滑块的index，默认为0
current-item-id	        String	当前所在滑块的item-id（不能同时指定current）

    index.wxml

    <!-- 标签页标题 -->
<view class="tab">
  <!--changeItem这个事件是单击按钮时进行切换 -->
  <view class="tab-item {{tab==0?'active':''}}" bindtap="changeItem" data-item="0">音乐推荐</view>
  <view class="tab-item {{tab==1?'active':''}}" bindtap="changeItem" data-item="1">播放器</view>
  <view class="tab-item {{tab==2?'active':''}}" bindtap="changeItem" data-item="2">播放列表</view>
</view>
<!-- 内容区域 -->
<view class="content">
  <swiper current="{{item}}" bindchange="changeTab">
    <swiper-item>
      <include src="info.wxml"/>
    </swiper-item>
    <swiper-item>
      <include src="play.wxml"/>
    </swiper-item>
    <swiper-item>
      <include src="playlist.wxml"/>
    </swiper-item>
  </swiper>

</view>
<!-- 底部播放器 -->
<view class="player">

</view>

    index.js

    Page({
  data: {
    item: 0,
    tab: 0
  },
  //导航栏tab改变时调用，使对应的item显示
  changeItem: function (e) {
    this.setData({
      item: e.target.dataset.item
    })
  },
  //item改变时调用，使导航栏对应改变
  changeTab: function (e) {
    console.log(e.detail)
    this.setData({
      tab: e.detail.current
    })
  }
})
